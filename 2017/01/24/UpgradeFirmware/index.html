Upgrade Firmware Tool for Captuvo

The UgradeFW Tool is iOS application tool for running in iOS device, which load correct firmware to upgrade Captuvo’s firmware.
Provide user to select which firmware to be upgrade into Captuvo device.

Please refer to the following resources for additional information regarding the Captuvo hardware and development tools:
How to use the UpgradeFW tool please refer the UpgradeFW Tool User Guide.
How to download it please refer apple appstore for search(will publish into appstore).
Honeywell Scanning and iOS website: http://www.honeywellaidc.com/
The UpgradeFW Tool support firmwares and mini firmware versions list as below:

UpgradeFW Tool auto recognize current iOS devices and load match current iOS device firmware for upgrade Captuvo.
